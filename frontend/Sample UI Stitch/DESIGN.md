# Design System Documentation

This document outlines the core visual principles and design tokens for our system.

## Theme Overview

Our design system is currently set to `light` color mode, ensuring a bright and clean user experience.

## Color Palette

We utilize a focused color palette to maintain brand consistency and visual hierarchy.

*   **Primary Color:** `#e43c29` - This is our brand's most distinctive color, used for key interactive elements and calls to action.
*   **Secondary Color:** `#EB8C00` - A supporting color for less prominent UI elements, chips, and secondary actions.
*   **Tertiary Color:** `#FFB600` - An additional accent color for highlights, badges, or decorative elements.
*   **Neutral Color:** `#7D7D7D` - Our base neutral color, ideal for backgrounds, surfaces, and non-chromatic elements.

## Typography

The system consistently uses the `inter` font family across all text roles for a unified and modern look.

*   **Headline Font:** `inter`
*   **Body Font:** `inter`
*   **Label Font:** `inter`

## Shape and Form

**Roundedness:** Our UI elements feature subtle roundedness (`1`), providing a slightly softer, yet still defined, aesthetic.

## Spacing

**Spacing:** We maintain a normal level of whitespace (`2`), balancing density with readability and visual comfort.
